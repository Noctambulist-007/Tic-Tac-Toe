# Tic Tac Toe Game

# Overview
This Flutter project is a Tic Tac Toe game with animations, built using the Dart programming language and Flutter framework.

# Features
1. Beautiful Animations
2. Responsive Design
3. User-Friendly Interface
4. Single player Modes
